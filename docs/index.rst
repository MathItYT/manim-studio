Welcome to Manim Studio's documentation!
========================================

Manim Studio is a graphical user interface for Manim (Community Edition).
It is designed to make the process of creating animations easier and more accessible for everyone,
enabling users to preview the animations in real-time while coding, interact with them, among other features.
